# OCRセットアップ

OCRは、画像やスキャンPDFに写っている文字を読み取る機能です。

このアプリでは、フォルダー内の画像やPDFを見つけることはできます。ただし、画像の中の文字を読むには、このPCにOCR用の道具が必要です。

## 使うもの

- Tesseract: 画像の文字を読むための本体
- tesseract-lang: 日本語などの言語データ
- Poppler: PDFを画像に変換してOCRに渡すための道具

## Macでの導入

プラグイン画面の「OCRをセットアップ」を押すと、Homebrewを使って必要なものを導入します。

初回は通信が発生します。目安は50MBから200MB程度ですが、環境や言語データで変わります。

導入が終わったら、プラグイン画面で「再確認」を押してください。「OCR: Tesseract」と表示されれば使えます。

## できること

- 画像ファイル内の文字を読む
- スキャンPDFや画像だけのPDFを数ページだけ読む
- 読み取った内容をチャットの根拠として使う

## できないこと

- 手書き文字を必ず正確に読む
- 画像が荒いPDFを完全に読む
- パスワード付きPDFを読む

## 注意

OCRはAIモデルではなく、PCに入れる読み取り用ツールです。Gemma 4 12BやQwenなどの言語モデルとは別に管理します。

Windows対応は今後追加予定です。
