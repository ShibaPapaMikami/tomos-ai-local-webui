# TOMOS MEMORY

## 目的

TOMOSが保存、参照、削除、忘却する情報の範囲を明確にする。

## 対象範囲

- チャット履歴
- キャラクター記憶
- 学習セット
- 教材パックの有効状態
- 契約書管理などのローカル台帳
- Local Context Coreに接続する将来機能

## 必須項目

- 保存対象
- 保存禁止
- 参照スコープ
- 忘れる条件
- ユーザー確認が必要な操作
- 保存先の種類
- 共有範囲

## 保存対象

- ユーザーが明示的に保存した修正例
- キャラクター設定
- キャラクター記憶として確認済みの情報
- 学習セット
- ローカル契約台帳の確認補助情報
- 教材パックの有効状態

## 保存禁止

- APIキー
- パスワード
- 認証トークン
- 個人番号、決済情報、医療情報
- 実在ユーザーや取引先の秘密情報を含むサンプル
- 未公開URL
- PCのローカルファイル全文の無断保存

## 忘れる条件

- ユーザーが削除を指示した場合
- 保存目的がなくなった場合
- 誤記憶と判定された場合
- スコープ外の情報だった場合
- 期限付き記憶の期限が切れた場合

## スコープ

- TOMOS本体: アプリ全体の標準記憶ルール
- フォルダー: 対象フォルダー内だけで参照
- チャット: 対象チャット内だけで参照
- キャラクター: 対象キャラクター会話だけで参照
- プラグイン: `PLUGIN.md` の `dataAccessScope` に従う

## 任意項目

- 保存期間
- ユーザー確認UI
- エクスポート形式
- ロールバック方法

## 禁止事項

- 個人情報や秘密情報をテンプレートに直接書かない。
- ユーザー確認なしに記憶を外部送信しない。
- クラウド同期を標準前提にしない。
- プラグインが許可範囲外のデータを読むことを許可しない。

## 記入例

```md
## 保存対象
- 修正して学習で保存した文章改善例

## 保存禁止
- APIキー
- 実在取引先の未公開情報

## スコープ
- フォルダー単位

## 忘れる条件
- ユーザーが削除した場合
```

## 変更時の確認方法

- 保存対象と保存禁止が両方書かれているか確認する。
- スコープがアプリ全体、フォルダー、チャット、プラグインのどれか分かるか確認する。
- 忘れる条件が明記されているか確認する。
