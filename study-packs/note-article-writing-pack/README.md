# note記事作成教材パック

noteに載せる記事の構成、本文、連載の続き、公開前確認を整えるための公開教材です。

## 使い方

1. `pack.json` と各 `promptFile` が入っている `note-article-writing-pack/` フォルダー全体を、TOMOSの教材パック取り込みで選択します。
2. 目的に合うモードを選び、下書きやメモを入力します。
3. 返ってきた本文を確認し、事実と個人情報を自分で確認してから公開します。

## 収録モード

- `rewrite-for-note`: 既存の下書きをnote向けの構成に整えます。
- `continue-series`: 既存の連載の文脈を保ち、次の記事の下書きを作ります。
- `paste-ready`: 貼り付けに必要な3項目だけを返します。
- `prepublish-check`: 公開前に確認する項目を整理します。

## ルール

- 公式仕様に沿う読みやすい見出し、段落、箇条書きを使います。
- 入力にない事実、数字、引用、体験、リンクは追加しません。
- 個人の名前、住所、パス、識別情報は一般化します。
- 例文は架空の題材だけを使います。

詳細は `note-format-rules.md` と `privacy-rules.md` を参照してください。
