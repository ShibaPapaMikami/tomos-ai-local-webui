# 教材パックのインポートガイド

教材パックは、配布教材、文章ルール、練習モード、例文、出典情報をまとめた読み取り用データです。
モデル本体を直接書き換えるものではありません。

## 教材パックと学習セットの違い

- 教材パック: 配布された基本ルール、プロンプト、モード、例文
- 学習セット: 利用者が保存した修正例、正しい回答、好み、追加ルール

教材パック本体は原則読み取り専用です。
「修正して学習」で保存した内容は、教材パックではなく学習セットに保存します。

## インポートできる形式

現時点では、`pack.json` を含むフォルダーをインポートできます。

```text
my-study-pack/
  pack.json
  README.md
  tone-guide.md
  modes/
    rewrite.md
    check.md
  examples/
    examples.md
```

ZIP形式の直接読み込みはまだ未対応です。
ZIPで配布された場合は、先に展開してからフォルダーを選択してください。

## pack.json

`pack.json` は教材パックの名前、説明、モードを定義するファイルです。

```json
{
  "id": "my-study-pack",
  "name": "自分用 文章ルール",
  "version": "0.1.0",
  "description": "自分の文章ルールに沿って整える教材パックです。",
  "visibility": "private",
  "modes": [
    {
      "id": "rewrite",
      "name": "文章を整える",
      "promptFile": "modes/rewrite.md"
    },
    {
      "id": "check",
      "name": "送信前チェック",
      "promptFile": "modes/check.md"
    }
  ]
}
```

## modes/*.md

`modes/*.md` には、AIに実行してほしい具体的な指示を書きます。

```markdown
# 文章を整える

入力された文章を、目的が分かりやすく、相手が次に何をすればよいか分かる形に整えてください。

## 重視すること

- 結論を先に書く
- 依頼事項を明確にする
- 事実を勝手に追加しない

## 出力形式

1. 修正版
2. 変更した理由
3. 確認事項
```

## private教材の扱い

社内資料や個人用ルールを含む教材は、`visibility` を `private` にしてください。

```json
"visibility": "private"
```

`private` の教材は、この端末にインポートして使う想定です。
公開サンプル教材の一覧へ入れないでください。

## 入れてはいけない情報

教材パックには、以下を入れないでください。

- 個人名
- 会社名、クライアント名
- 金額、契約情報、権利関係
- 未公開プロジェクト名
- URL
- APIキー
- パスワード
- 個人情報
- 実メール、実Slackの未匿名化本文

迷う情報は入れず、まず社内確認してください。

## インポート手順

1. 教材パックフォルダーを用意する
2. アプリの「教材パック」を開く
3. 「外部教材データをインポート」を押す
4. `pack.json` が入っているフォルダーを選ぶ
5. 追加された教材パックを有効にする
6. チャット画面の教材モードから使う
