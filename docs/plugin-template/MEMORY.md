# Plugin MEMORY Template

## 目的

このプラグインが保存・参照・削除できる情報の範囲を定義する。

## 対象範囲

- 保存する情報:
- 参照する情報:
- 削除する情報:
- 対象スコープ:

## 必須項目

- 保存対象
- 保存禁止
- 参照スコープ
- 忘れる条件
- ユーザー確認が必要な操作
- 保存先

## 任意項目

- 保存期間
- エクスポート形式
- 復元方法
- 監査ログ方針

## 禁止事項

- APIキー、パスワード、認証トークンを保存しない。
- 個人情報や秘密情報をサンプルに書かない。
- ユーザー確認なしに外部送信しない。
- `PLUGIN.md` の `dataAccessScope` 外の情報を参照しない。

## 記入例

```md
## 保存対象
- ユーザーが確認した下書き

## 保存禁止
- APIキー
- 秘密情報

## スコープ
- 選択中フォルダーのみ

## 忘れる条件
- ユーザーが削除した場合
```
