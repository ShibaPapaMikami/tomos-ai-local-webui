# Plugin AGENTS Template

## 目的

このプラグインをAIエージェントが安全に実装・調査・検証するためのルールを書く。

## 対象範囲

- pluginId:
- 対象ファイル:
- 触ってよい範囲:
- 触らない範囲:

## 必須項目

- 作業境界
- 安全ルール
- 検証方法
- 依存追加の可否
- 外部API実行の可否
- 本番変更の可否

## 任意項目

- 担当者
- 既知の制約
- ロールバック方法
- 手動確認手順

## 禁止事項

- 個人情報、APIキー、秘密情報を書かない。
- TOMOS本体仕様を無断で変更しない。
- 既存プラグインの仕様を無断で変更しない。
- Agent-Reach本体やInternet Layerへこのテンプレートから介入しない。

## 記入例

```md
## 触ってよい範囲
- docsのみ

## 触らない範囲
- TOMOS本体コード
- Agent-Reach本体コード
- 外部API連携

## 検証
- Markdown確認
- git diff --check
```
