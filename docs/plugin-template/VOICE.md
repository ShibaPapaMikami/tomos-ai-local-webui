# Plugin VOICE Template

## 目的

このプラグインが出力する文章、会話、通知、キャラクター口調を定義する。

## 対象範囲

- 通常返信:
- キャラクター返信:
- 通知:
- エラー:

## 必須項目

- 呼び方
- 一人称
- 文体
- 文量
- 確認質問の扱い
- NG表現
- リライト時の形式

## 任意項目

- 口調サンプル
- 状況別テンプレート
- キャラクター設定との優先順位

## 禁止事項

- 実在ユーザーや取引先の情報をサンプルに書かない。
- AIが実在の人間であるように装わない。
- 専門判断を断定しない。
- 安全性を下げる口調指示を優先しない。

## 記入例

```md
## 文体
- 短く、やさしく、事実不明時は確認する

## NG表現
- 断定できない事実の断定
- 外部送信済みのように見える表現
```
