# 手動確認チェックリスト

リファクタリング後に、主要な動作が壊れていないか確認するための一覧です。毎回すべてを確認する必要はありません。変更した範囲に近い項目を優先します。

## 学生向けモデル

- [ ] `標準AI` の Qwen3 4B が、チャット、翻訳、資料検索、学習に使われる
- [ ] 翻訳専用カードが重複せず、翻訳は標準AIへ統合されている
- [ ] `コード作業` は Agentic Coder v2 が取得済みの場合だけ使われ、未取得時は標準AIへ戻る
- [ ] `高性能AI` の Gemma 4 12B は任意の追加候補で、自動の標準にならない
- [ ] Qwen3 4B が未取得の12GB未満PCだけ、Qwen 2.5 3B が予備の軽量AIとして案内される
- [ ] 旧設定の非表示モデルだけが安全な自動選択へ移行し、Gemma、Qwen 2.5、Qwen3 4B、Agentic Coderの既存選択は維持される
