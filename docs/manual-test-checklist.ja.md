# 手動確認チェックリスト

リファクタリング後に、主要な動作が壊れていないか確認するための一覧です。毎回すべてを確認する必要はありません。変更した範囲に近い項目を優先します。

## 起動

- [ ] `Gemma4_12B_全部起動.command` で起動できる
- [ ] ブラウザで `http://127.0.0.1:54876/` が開く
- [ ] 左下の状態が「使用可能」になる
- [ ] 設定を開閉できる
- [ ] バージョン情報が表示される

## チャット

- [ ] 「おはよ」に短く自然に返る

- [ ] Enterは改行、送信ボタンで送信できる
- [ ] Cmd/Ctrl + Enterで送信できる
- [ ] 応答モードをフォームから変更できる
- [ ] モデル自動/モデル選択をフォームから変更できる
- [ ] ユーザー発話とGemma回答の全文コピーができる

## Web検索・外部リンク

- [ ] Web検索をON/OFFできる

- [ ] 参照リンク `[1]` `[2]` に外部リンクアイコンが付く
- [ ] 参照リンクを開くと別タブで外部サイトが開く
